# Pitchfork Pro

یک اندیکاتور پیشرفته برای MetaTrader 5 که چنگال اندروز (Pitchfork) را با قابلیت‌های سفارشی‌سازی کامل رسم، ذخیره و مدیریت می‌کند.

## 📋 ویژگی‌ها

- **رسم خودکار Pitchfork**: اسکن و شناسایی Pitchforkهای رسم شده روی چارت
- **ذخیره‌سازی پایدار**: حفظ تنظیمات و موقعیت‌ها پس از راه‌اندازی مجدد متاتریدر
- **خطوط میانی اضافی**: رسم خطوط فیبوناچی و میانی بین خطوط اصلی
- **جایگزینی هوشمند**: جایگزینی Pitchfork استاندارد MT5 با نسخه سفارشی
- **مدیریت چندگانه**: پشتیبانی از چندین Pitchfork همزمان روی یک چارت
- **تشخیص جهت**: شناسایی خودکار روند صعودی یا نزولی
- **داشبورد اطلاعات**: نمایش وضعیت و اطلاعات Pitchforkها
- **لاگ‌گیری پیشرفته**: سیستم لاگ‌گیری برای دیباگ و مانیتورینگ

## 🎯 کلیدهای میانبر

| کلید | عملکرد |
|------|--------|
| `S` | اسکن، ذخیره و رسم Pitchfork |
| `R` | جایگزینی Pitchfork اصلی با نسخه سفارشی |
| `D` | حذف Pitchfork انتخابی |
| `G` | نمایش/مخفی کردن داشبورد |

## 🚀 نصب

1. تمام فایل‌های پروژه را در مسیر زیر کپی کنید:
   ```
   [مسیر نصب MetaTrader 5]/MQL5/Indicators/PitchforkPro/
   ```

2. متاتریدر 5 را مجدداً راه‌اندازی کنید.

3. اندیکاتور `PitchforkPro.mq5` را از پنجره Navigator به چارت مورد نظر بکشید.

## 💡 نحوه استفاده

1. **رسم Pitchfork استاندارد**: ابتدا با استفاده از ابزار Pitchfork متاتریدر، سه نقطه (A, B, C) را روی چارت مشخص کنید.

2. **فعال‌سازی**: کلید `S` را فشار دهید تا اندیکاتور Pitchfork را اسکن و نسخه سفارشی را رسم کند.

3. **جایگزینی**: در صورت نیاز، کلید `R` را فشار دهید تا Pitchfork اصلی با نسخه سفارشی جایگزین شود.

4. **حذف**: برای حذف Pitchfork انتخابی، کلید `D` را فشار دهید.

5. **داشبورد**: برای نمایش/مخفی کردن داشبورد اطلاعات، کلید `G` را فشار دهید.

6. **ذخیره‌سازی**: تمام تنظیمات به صورت خودکار ذخیره شده و پس از راه‌اندازی مجدد قابل بازیابی هستند.

## ⚙️ پارامترها

- **EnableAutoReplace**: فعال/غیرفعال کردن جایگزینی خودکار
- **ShowMidLines**: نمایش خطوط میانی اضافی
- **ShowDashboard**: نمایش داشبورد اطلاعات
- **LineColor**: رنگ خطوط
- **LineWidth**: ضخامت خطوط
- **LineStyle**: سبک خطوط
- **LogLevel**: سطح لاگ‌گیری (NONE, ERROR, WARN, INFO, DEBUG)

## 🔧 توسعه

### پیش‌نیازها
- MetaTrader 5
- MQL5 IDE (اختیاری)

### کامپایل
1. فایل `PitchforkPro.mq5` را در MetaEditor باز کنید.
2. دکمه Compile را فشار دهید (F7).
3. در صورت وجود خطا، فایل‌های `.mqh` نیز باید کامپایل شوند.

### دیباگ
- از MetaEditor برای دیباگ استفاده کنید.
- لاگ‌ها در تب \"Experts\" متاتریدر قابل مشاهده هستند.
- سطح لاگ‌گیری را در پارامترها تنظیم کنید.

### تست
- فایل `PitchforkPro_Tester.mq5` را برای تست قابلیت‌ها اجرا کنید.
- گزارش‌های تست در پوشه `MQL5/Files/PitchforkPro/` ذخیره می‌شوند.
